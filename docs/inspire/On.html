§ The On valosheath namespace of the library ontology of @valos/inspire

The ValOS inspire On namespace contains event callback names used by the inspire UI layer.

The namespace inherits all HTML5 event names verbatim as name suffixes but also adds new valos-specific event callback names. Like HTML5 events these callbacks are called with a synthetic event as their first argument.
